# 2_reads — Read-Level Microbiome Analysis Summary

This directory contains summary-level outputs from read-based metagenomic analyses, organized in a single Excel file.

---

## File

### `Reads_summary.xlsx`
This Excel workbook contains multiple sheets summarizing read-level taxonomy, diversity, and comparative analyses.

---

## Sheets Overview

### 1. Table S1
- General summary table of read statistics and sample metadata.

---

### 2. Reads_minimap_kaiju
- Taxonomic annotation of reads using minimap alignment and Kaiju-based classification.
- Provides read-level taxonomic assignments.

---

### 3. Reads_species_NMDS
- NMDS (Non-metric Multidimensional Scaling) analysis at species level.
- Used to visualize community similarity across samples.

---

### 4. Reads_venn
- Venn diagram data for shared and unique reads across samples or groups.

---

### 5. Reads_phylum
- Taxonomic composition of reads summarized at the phylum level.

---

### 6. NCBI_colony
- Read assignments or comparisons against NCBI reference database (colony mosquito test: adaptive vs. non-adaptive). 

---

### 7. NCBI_H
- Read assignments or comparisons against NCBI reference database (for hamden group) 

---

### 8. NCBI_G
- Read assignments or comparisons against NCBI reference database (for Guilford group) 

---

### 9. NCBI_T
- Read assignments or comparisons against NCBI reference database (for Tau group) 

---

## Notes

- This dataset focuses on **reads-level microbial profiling**, rather than assembled genomes.
