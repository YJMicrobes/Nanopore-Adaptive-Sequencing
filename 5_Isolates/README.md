# 5_Isolates — Isolate-Based Microbial Reintroduction Experiment

This directory contains analyses related to microbial isolate experiments, including isolate abundance tracking, taxonomic annotation, and experimental validation of microbial reintroduction effects in mosquito culture systems.

---

## Overview

This module focuses on isolate-level ecology and experimental microbiology, specifically:

- Tracking isolate abundance across samples  
- Taxonomic annotation of isolates using GTDB-Tk  
- Experimental evaluation of microbial reintroduction effects  
- Functional/ecological interpretation of isolate addition experiments  

---

## Experimental Context (HATCH2026)

The **HATCH2026 experiment** investigates the effect of reintroducing cultured bacterial isolates into mosquito microbiomes.

- Selected isolates (e.g., *Bacillus* spp. and *Comamonas* spp.) were reintroduced into mosquito culture systems.
